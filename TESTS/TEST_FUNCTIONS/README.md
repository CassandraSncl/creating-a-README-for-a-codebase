- **Description** : Calcule le carré d'un nombre.

  - `x` (int/float) : Le nombre à élever au carré.

## `random_between(a, b)`
- **Description** : Génère un nombre entier aléatoire entre deux bornes.
- **Paramètres** :
  - `a` (int) : Borne inférieure (inclusive).
  - `b` (int) : Borne supérieure (inclusive).
- **Retour** : Un entier aléatoire entre `a` et `b`.

## `circle_area(radius)`
- **Description** : Calcule l'aire d'un cercle à partir de son rayon.
- **Paramètres** :
  - `radius` (int/float) : Le rayon du cercle.
- **Retour** : L'aire du cercle (π * radius²).
